# ScrumMaster AI

An AI-powered Scrum Master extension that allows developers to directly convert intent and code context into high-quality Jira work items with minimal friction.

## Features
* **1-Click Code Extraction:** Automatically grabs your active file name and highlighted code to provide context to the AI.
* **AI Ticket Generation:** Drafts complete Jira tickets (Title, Description, ACs, Story Points) using AWS Bedrock.
* **Frictionless UI:** Operates natively in the VS Code Activity Bar so you never have to leave your IDE.

## Usage
1. Click the Robot icon in the Activity Bar.
2. Highlight the code you want to work on and click "Extract from Active Code".
3. Add any manual instructions to the text box.
4. Click "Create Jira Ticket".
